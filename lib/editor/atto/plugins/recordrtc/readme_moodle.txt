mp3-mediarecorder v4.0.5

Library update instructions:

The files here are the built version of the MIT Licensed https://github.com/elsmr/mp3-mediarecorder,
saved from their CDN.

Copy the dist/ and worker/ files to the correct place.

    1. https://unpkg.com/mp3-mediarecorder@4.0.5/dist/index.umd.js -> amd/src/local/mp3-mediarecorder/mp3-mediarecorder.js

    2. https://unpkg.com/mp3-mediarecorder@4.0.5/worker/index.umd.js -> amd/src/local/mp3-mediarecorder/mp3-mediarecorder-worker.js

    3. https://unpkg.com/vmsg@0.4.0/vmsg.wasm -> amd/src/local/mp3-mediarecorder/vmsg.wasm
